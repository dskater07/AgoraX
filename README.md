# AgoraX
AgoraX es un proyecto académico del ITM en la materia Calidad del Software. Desarrolla un sistema de votación electrónica para asambleas de conjuntos residenciales, basado en FastAPI, Vue.js y Docker, enfocado en calidad, trazabilidad, seguridad, transparencia y cumplimiento de la Ley 675 de 2001.
